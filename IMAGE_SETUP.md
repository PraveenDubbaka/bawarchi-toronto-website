# Image Assets Setup

Please save the following images to complete the website setup:

## Required Images:

### 1. Hero Banner Image (Spices)
- **Source**: The image with colorful Indian spices on wooden spoons
- **Save as**: `public/images/spices-hero.jpg`
- **Description**: This will be used as the hero banner background

### 2. Bawarchi Logo
- **Source**: The Bawarchi Biryanis logo image
- **Save as**: `public/images/bawarchi-logo.png`
- **Description**: This will be used in the navigation bar and footer

### 3. Food Images (Download from free sources)

You can download high-quality Indian food images from these sources:
- **Unsplash**: https://unsplash.com/s/photos/indian-food
- **Pexels**: https://www.pexels.com/search/indian%20food/
- **Pixabay**: https://pixabay.com/images/search/indian%20food/

#### Required Food Images:

1. **Biryani Image**
   - Save as: `public/images/biryani.jpg`
   - Search for: "Hyderabadi Biryani" or "Indian Biryani"
   - Should show colorful rice dish with garnish

2. **Paneer Tikka Image**
   - Save as: `public/images/paneer-tikka.jpg`
   - Search for: "Paneer Tikka" or "Grilled Paneer"
   - Should show grilled cottage cheese cubes

3. **Butter Chicken Image**
   - Save as: `public/images/butter-chicken.jpg`
   - Search for: "Butter Chicken" or "Chicken Tikka Masala"
   - Should show creamy orange curry

4. **Tandoori Image**
   - Save as: `public/images/tandoori.jpg`
   - Search for: "Tandoori Platter" or "Tandoori Chicken"
   - Should show grilled/tandoor items

## Quick Setup Instructions:

1. Create the `public/images` folder if it doesn't exist
2. Download the images from the sources above
3. Rename them according to the list above
4. Place them in the `public/images` folder
5. Refresh your browser
